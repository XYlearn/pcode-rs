pcode-rs is a library built around the SLEIGH library.

SLEIGH and the processor definition files under the pcode-rs/processors directory
originate from the Ghidra project (https://ghidra-sre.org/). SLEIGH is released
under the terms of the Apache 2 license as defined in pcode-rs/cpp/LICENSE. See
NOTICE file in pcode-rs/cpp/NOTICE.

pypcode is also a python library built around the SLEIGH library. pcode-rs modifies
the wrapper code and build script from pypcode under cpp. pypcode is released under
the 2-clause BSD license as defined in cpp/pypcode_LICENSE.

The remaining code of pcode-rs, unless stated otherwise, is licensed under the
terms of the 2-clause BSD license below.
